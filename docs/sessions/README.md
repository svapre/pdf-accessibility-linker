# Session Logs

Use this folder to record execution evidence for changes covered by AI settings enforcement.

## Why this exists
- Keeps a machine-checkable trail of:
  - planned changes,
  - user approval evidence,
  - failures seen,
  - corrective actions taken before retries.

## Naming
Use: `YYYY-MM-DD-short-topic.md`

Example:
- `2026-02-24-ai-settings-and-session-enforcement.md`

## Template
Start from `docs/sessions/TEMPLATE.md`.
